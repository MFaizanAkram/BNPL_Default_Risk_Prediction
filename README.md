# Buy Now Pay Later (BNPL) Default Risk Prediction

This project predicts the probability of default for BNPL users using a machine learning model.
It is built with **Python, XGBoost, and Streamlit** for an interactive risk assessment interface.
Users input customer and transaction details to get default probability, risk category, and decision.
The app also explains predictions using **SHAP values** for transparency.
Two main files:

* `project.ipynb` → data preparation, feature engineering, model training
* `app.py` → Streamlit app for deployment and inference
  Designed for academic and practical fintech risk analysis use cases.
